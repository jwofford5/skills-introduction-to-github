02 — Long-Term Memory

The Unlimited Store

Long-term memory is effectively unlimited in both capacity and duration. It holds everything you know — language, faces, skills, facts — organised into schemas: mental structures that bundle related pieces into a single retrievable unit.

A chess grandmaster doesn't see 32 individual pieces; they see formations — each one a schema built over years. That bundling is the engine of expertise. The ultimate goal of any instruction is schema formation in long-term memory.

"Once appropriate information is stored in long-term memory, the capacity and duration limits of working memory are transformed and indeed, humans are transformed."

— Sweller, Ayres & Kalyuga, Cognitive Load Theory (2011)
03 — Working Memory

The Bottleneck

Working memory is where conscious thought happens — you are only aware of what is currently in it. But when handling novel information, it is severely constrained in two critical ways:

~4
Items — Storage
Earlier estimates were 7±2 (Miller, 1956). Cowan (2001) found ~4 items is more representative for active processing.
2–3
Elements — Processing
When novel elements must be organised or compared, the usable limit drops even below storage capacity.
<20s
Duration — Without Rehearsal
Novel information fades in under 20 seconds unless actively rehearsed (Peterson & Peterson, 1959).

These are not design flaws. With just 3 novel elements there are 6 possible permutations; with 10 there are 3.6 million. A small working memory keeps that combinatorial explosion manageable when facing the genuinely unknown.

04 — Dual Channel

Two Processors, Not One

Working memory isn't a single bucket. Baddeley's model identifies two partially independent processors — one for each sensory modality. Using both simultaneously can expand effective capacity.

Visual 👁

Visual-Spatial Sketchpad

Handles images, diagrams, spatial layouts, and written text perceived visually.

Auditory 🔊

Auditory / Phonological Loop

Handles spoken words, sounds, and inner speech when mentally "sounding out" text.

A diagram paired with spoken narration splits the load across both channels. The same diagram with identical on-screen text forces everything through the visual channel and can overload it.

05 — The Expert Advantage

When the Limits Disappear

The capacity and duration limits apply only to novel information. When working memory draws on schemas already stored in long-term memory, those constraints effectively vanish.

A schema — however complex — counts as a single element. An expert radiologist reading an X-ray isn't processing hundreds of individual features; they retrieve a handful of high-level patterns as single units. The working memory hasn't grown; long-term memory is doing the heavy lifting.

06 — Practical Examples

Theory in the Real World

📞
Duration limit

Dialling an Unfamiliar Number

You look up a number, walk to the phone — and it's gone. That's the ~20-second duration limit. You solve it by rehearsing aloud, keeping the digits alive in the phonological loop. A number you call daily needs no rehearsal: a long-term memory schema retrieves it instantly.

🚗
Processing limit + schema formation

Learning to Drive vs. Driving as an Expert

A new driver consciously tracks steering, mirrors, pedals, speed, and traffic simultaneously — often hitting the 2–3 element processing cap. An experienced driver has bundled those sub-tasks into automatic schemas, freeing working memory to hold a conversation or react to the unexpected.

♞
Schemas as chunks

Novice vs. Expert Chess Player

Chess masters briefly shown a mid-game board recalled nearly every piece; beginners only a few. Shown a random arrangement, masters were no better than beginners. Experts don't have bigger working memories — they have thousands of board-position schemas, each recalled as one chunk.

🗺
Dual channel

Map + Spoken Directions

Reading a road map while a passenger reads directions aloud is manageable — one task per channel. Reading the directions on-screen while studying the map forces both into the visual channel, rapidly overloading it and degrading performance on both tasks.

07 — Instructional Takeaways

What This Means for Teaching

  • 1 Respect the bottleneck. Novice learners can process no more than 2–3 novel concepts at once. Break complex material into small sequential steps rather than presenting everything simultaneously.
  • 2 Build schemas deliberately. Every lesson should aim to move organised knowledge into long-term memory. Worked examples, guided practice, and spaced retrieval accelerate schema formation — and directly expand the learner's functional capacity.
  • 3 Use both channels. Pair visual content (diagrams, animations) with spoken explanation rather than on-screen text. Distributing load across the visual and auditory processors avoids overloading either one.
